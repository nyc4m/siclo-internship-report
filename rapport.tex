\abstracttext{%
    Durant mon stage à Siclo Mobile, j'ai eu l'occasion de travailler sur trois
    applications côté serveur.

    L'enjeux pour ces trois projets étaient soit de :
    \begin{itemize}
        \item définir une API pour permettre à une application
            de pouvoir communiquer avec un serveur pour récupérer des données
            ou en créer.
        \item définir des microservices afin de pouvoir effectuer un traitement continue sur 
            un flux de données provenant d'objets connectés.
    \end{itemize}

    Ces projets on été développé en utilisant deux langages populaires du web: \textit{php et nodejs}.
